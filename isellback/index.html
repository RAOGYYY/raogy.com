Startup Architecture

Inside iSellBack

A multi-layered recommerce system powered by secure AWS infrastructure, dynamic ML pricing engines, and real-time agent dispatch mechanisms.

Customer App

A web and mobile checkout system where customers select their MacBook/iPad, answer condition questions, and generate an instant quote.

  • Instant pricing engine (<60s)< /li>
  • Live device condition questionnaire
  • Seamless doorstep pickup schedule

Field Agent App

A dedicated platform (`field.buybackelite.com`) for verified collection agents to locate orders, verify hardware specifications, and execute payouts.

  • Real-time doorstep agent tracking
  • On-the-spot physical state validation
  • Secure on-premise data sanitation

Admin Console

An administrative control panel (`control.buybackelite.com`) to manage the system parameters, track business goals, adjust price equations, and dispatch agents.

  • Custom API synchronization (e.g. Cashify)
  • Pricing margin controls & adjustments
  • Comprehensive database monitoring

01 Security & Infrastructure

AWS Infrastructure Scaling

iSellBack runs on a highly secure and optimized server layout. Using serverless application handlers, API calls scale dynamically based on active requests. Static content is cached on a global content delivery network, ensuring sub-second response times.

Supabase Backend

Database rows are shielded behind granular Row-Level Security (RLS) policies. User authentication is verified via OAuth integration protocols, ensuring safe transaction processes from initial sign-up to payout approval.

Technical Highlights

Real-Time Messaging

Firebase Cloud Messaging alerts agents and customers on status adjustments.

Dynamic Pricing Feeds

APIs synchronizing with pricing platforms like Cashify keep valuations precise.

Secure Data Deletion

Verified diagnostic checklists ensure devices undergo complete factory reset prior to packaging.

03 Operational Workflow

1

Submit Request & Quote Validation

Users query quotes on iSellBack. The system parses structural values and assigns a temporary valuation index.

2

Doorstep Verification

Collection agent completes physical state checks on-premise, adjusting specifications to determine local market worth.

3

Instant Payout Processing

The agent confirms the transaction status. Supabase triggers database webhooks that run fast secure transfers (UPI/Direct Bank Transfer) instantly.
